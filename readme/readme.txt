Formatters for lexers CSS/SCSS/SASS/LESS.

- "CSS Beautify": formats CSS (only CSS lexer allowed) using engine cssbeautifier (from jsbeautifier.org).
- "CSS Expand", "CSS Compact", "CSS Compress": formats CSS/SCSS/SASS/LESS using alternative engine cssformatter. https://github.com/mutian/CSS-Formatter
- "CSS Compressor": minifies CSS code using engine csscompressor. https://pypi.org/project/csscompressor/
